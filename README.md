# 🦅 Соколов AI

Интеллектуальный чат-ассистент на базе DeepSeek-V3.2 с поддержкой голосового ввода, анализа кода и потоковой передачи ответов.

## 🚀 Возможности

- 💬 **Чат с AI** — интеллектуальные ответы на любые вопросы
- 🎤 **Голосовой ввод** — диктуйте вместо набора (работает на iOS, Android, Windows)
- 📁 **Анализ кода** — загружайте файлы для анализа и рефакторинга
- 📱 **Адаптивный дизайн** — идеально работает на всех устройствах
- 🔄 **Потоковая передача** — ответы появляются в реальном времени
- 💾 **Сохранение истории** — все чаты сохраняются локально
- 🎨 **Современный UI** — стильный темный интерфейс
